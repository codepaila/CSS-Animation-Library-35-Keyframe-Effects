This repository contains **35 hand-crafted CSS animations** using `@keyframes`, `transform`, and various timing functions. Each animation is:

- ✅ **Ready to use** — copy and paste
- ✅ **Well documented** — comments explain every property
- ✅ **Responsive** — works on all screen sizes
- ✅ **Lightweight** — no JavaScript required
- ✅ **Cross-browser compatible** — Chrome, Firefox, Safari, Edge
